Итак, сегодня твой выпускной экзамен, палиндромный курсант. Пора приложить руку к чему-нибудь по-настоящему стоящему. Например, выведи палиндромную длину каждого префикса данной строки.

Входные данные

Дана строка s (1 ≤ |s| ≤ 106), состоящая из строчных латинских букв.

Выходные данные

Выведите n чисел, разделённых пробелом, — палиндромную длину для префикса s длины 1, 2, ..., n соответственно.
